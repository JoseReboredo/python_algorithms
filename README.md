
## Learnings
| Situation                | Data structure |
| ------------------------ | -------------- |
| Insert/remove from front | `deque`        |
| Lookup by key            | `dict`         |
| Unique items             | `set`          |
| Frequency counts         | `Counter`      |

## Most common patterns

### HashMap -> dictionaries

Used for:
- Two sums
- Frequency counting
- Fast lookups

### Sliding Window

Used for:
- moving averages
- longest substring
- rate limiting
- log analysis

### Two Pointers

Used for:
- sorted arrays
- interval problems
- deduplication

### Interval Merging

Used for:
- meeting scheduling
- booking conflicts
- time ranges

## O()
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ)
